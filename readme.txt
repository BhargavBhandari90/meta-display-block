A Gutenberg block to display post meta fields with optional SVG icon.

== Description ==

A customizable Gutenberg block to showcase post meta fields with dynamic labels, icons, and styling.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add/Edit post and search for "Meta Display" block
